# Name Hat

Given a group of people, randomly assign a unique member to another member of the group.

## Example `src/members.js` file

```js
module.exports = [
    {
        ID: 0,
        name: 'Person A',
        mobile: '+61000000000',
    },
    {
        ID: 1,
        name: 'Person B',
        mobile: '+61000000000',
    },
    {
        ID: 2,
        name: 'Person C',
        mobile: '+61000000000',
    },
]
```
